## 📌 Overview

**Toorism** is a sustainability-driven tourism platform that empowers travelers to make **eco-conscious decisions**. By integrating **carbon footprint tracking**, **route optimization**, and **smart recommendations**, Toorism transforms how people travel — from convenience-first to **impact-aware**.

Built for the **International Innovation Challenge 2k26**, this project directly addresses global sustainability challenges through technology.

---

## 🚨 Problem Statement

Tourism contributes significantly to global carbon emissions, yet travelers lack simple, accessible tools to measure and reduce their environmental impact.

---

## 💡 Proposed Solution

Toorism introduces a **data-driven travel assistant** that enables users to:

* 🌱 Understand their **carbon footprint in real time**
* 🧭 Choose **eco-friendly travel alternatives**
* 📊 Make **informed, sustainable decisions**

---

## ✨ Key Features

| Feature                | Description                                                |
| ---------------------- | ---------------------------------------------------------- |
| 🌱 Carbon Calculator   | Estimates emissions from transport, routes, and activities |
| 🗺️ Route Optimization | Uses OSRM to find efficient & low-emission routes          |
| 🧭 Eco Recommendations | Suggests greener travel options                            |
| 📊 Dashboard           | Visualizes environmental impact                            |
